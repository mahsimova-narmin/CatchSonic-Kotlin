#CatchSonic 

A simple Android game built with Kotlin.

The goal is to tap the visible character before time runs out and achieve the highest score possible.

## Features
- CountDownTimer-based game loop
- Score tracking
- High score persistence using SharedPreferences
- AlertDialog for restart logic
- Random image visibility logic
- Results screen

## Built With
- Kotlin
- Android SDK
- ViewBinding
- CountDownTimer
- SharedPreferences
- AlertDialog
